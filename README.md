# hercules

Remote terminal control

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Edit .env according to you

### Installing

```
make build
```

#### For server

```
cd server
./server
```

#### For terminal

```
cd implant
./implant
```

## Example

```
cd client
./client ls
```
